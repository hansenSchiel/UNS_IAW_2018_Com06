Información

El sitio está construido en base a un template gratuito basado en boostrap. Se utilizan las herramientas HTML, CSS, JS (y jQuery).
Existe la posibilidad de cambiar el tema del sitio desde la página principal, mediante el menu "Cambiar estilo". El sitio utiliza cookies para recordar el tema elegido por el usuario.
Los datos utilizados son estáticos, y las pantallas son a modo de ejemplo de la estrucura básica del sitio.
